Example: Sum of Even Numbers

  • Add all even numbers
  • Up to the given limit
  • What's the time complexity?
    • \[O(n)\]
                
                  def sum_of_even_integers(limit):
                      sum = 0
                      for any_integer in range(limit):
                          if any_integer % 2 == 0:
                              sum += any_integer
                      return sum
